# 02 单篇论文知识卡片生成提示词

## 用途

将单篇论文的摘要/全文转化为结构化知识卡片。

---

## 提示词

```
请为以下论文生成知识卡片，使用指定模板格式。

论文信息：
- 标题：
- 作者：
- 期刊/会议：
- DOI / arXiv ID：
- 全文是否可获取：

要求：
1. 按 templates/01_论文知识卡片模板.md 格式输出。
2. 重点提取"可迁移到 GIS/遥感/空间分析 的点"。
3. 如果论文方法是通用方法（如时序验证、消融实验），可标注"可附带借鉴到 ETF 模型"。
4. 如果只读了摘要，在卡片开头标注"仅基于摘要，结论未完全确认"。
5. 写作句式部分：提取论文中的经典句式、学术表达、逻辑链。
6. AI Code 任务部分：如果方法可复现为代码，生成任务包格式（使用 templates/02_AI_Code任务模板.md）。
7. 不确定的细节标注"未确认"。
8. 输出全部中文（论文标题、DOI 和技术术语保留原文）。
```
