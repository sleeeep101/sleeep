## 筛选流程（5步）

### 第1步：方向匹配（0-5分）

| 方向 | 说明 |
|------|------|
| 数字地形分析（DTA） | DEM数据处理、地形因子提取、地形分类 |
| DEM地貌形态 | 地貌形态识别、地形特征提取、地貌演化模拟 |
| 土壤侵蚀 | RUSLE模型、侵蚀沟识别、水土流失评估 |
| GIS空间分析与空间建模 | 空间统计、空间插值、地统计、适宜性评价 |
| 水土保持 | 水土保持措施、植被恢复、生态修复 |
| 地图学与地理信息系统 | 地图制图、空间数据模型、GIS系统设计 |
| 荒漠化防治 | 荒漠化监测、土地退化评估 |

### 第2步：方法迁移价值（0-5分）

### 第3步：期刊/来源质量加权

| 期刊层次 | 系数 |
|----------|:---:|
| Nature/Science | ×1.5 |
| Nature子刊/RSE/ISPRS/IJGIS/Geomorphology/Catena | ×1.2 |
| 中科院一区/TOP | ×1.1 |
| 其他SCI | ×1.0 |
| 中文核心 | ×0.9 |
| arXiv预印本/会议 | ×0.7 |

### 第4步：分级

综合分 = 方向匹配×1.5 + 方法迁移×1.0×期刊系数

| 综合分 | 分级 | 行动 |
|--------|:---:|------|
| ≥10 | **A级** | 精读全文 + 完整知识卡片 + ≥60分入KB |
| 6-9 | **B级** | 读摘要+方法 + 提取方法卡片 + ≥50分入KB |
| 3-5 | **C级** | 读摘要 + 归档 + 不入KB |
| <3 | **X级** | 跳过，不记录 |

### 第5步：加减分

**加分（各+1）**：中国典型地貌区/开源代码+公开数据集/传统GIS+ML结合/DEM为主要输入
**减分（各-1）**：纯理论无数据验证/仅摘要无全文/高收费墙无预印本/非中英文

## 在流水线中的位置

```
桌面PDF → 预筛(A/B/C/X分级) → 去重 → 文本提取 → 精读评分 → 知识卡片 → 日报/KB
          ↑ 新增步骤（2026-07-13）
```

- A级论文全部走完整精读流程
- B级论文走简化流程（方法卡片为主）
- C级论文仅摘要归档，不触发精读Agent
- X级论文直接跳过
