# 临时运行提示词 — 日报生成保险词

## 用途

在 `daily_paper_curator.py` 修改生效前，将此提示词直接塞入日报生成 prompt 作为过渡保险，
确保摘要级论文不会被伪装成精读。

---

## 提示词

```
生成今天的论文阅读日报时，请严格区分"摘要级阅读"和"全文精读"。

如果没有成功下载并解析 PDF 正文，不允许写"精选精读""主要结论""创新点""局限""正式学术评分""可复现任务"。只能写"摘要级速览"，并在每篇论文中明确标注：

* 阅读等级：ABSTRACT_ONLY
* 是否已读全文：否
* 证据来源：标题/摘要/元数据
* 待全文确认：方法、数据、结果、局限、可复现性
* 是否可入长期知识库：否，进入待补读队列

一句话总结必须以"从摘要看"或"摘要显示"开头。
所有方法、数据、结论都必须使用"可能""待确认""摘要未提供"等保守措辞。

只有当 PDF 正文已被成功读取，并覆盖引言、方法、结果、讨论/结论等关键部分时，才允许标注为 PDF_TEXT_FULL，并生成全文精读卡片。
```
