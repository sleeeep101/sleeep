## 答辩委员会20个尖锐问题（8个维度）

| 维度 | 典型问题 |
|------|----------|
| 选题意义 | 为什么选这个题目？这个方向的前沿在哪里？为什么研究川东北而不是其他区域？ |
| 创新性 | 你的方法和已有方法到底区别在哪里？是真的创新还是换数据跑了一遍？ |
| 数据质量 | DEM分辨率是否足够？SRTM 30m在丘陵区的误差对结果的影响？ |
| 方法选择 | 为什么选CNN而不选随机森林/Transformer？深度学习相比传统方法的增益是否值得其复杂度？ |
| 实验设计 | 如何保证训练和测试的空间独立性？基线选择是否合理？消融实验是否充分？ |
| 统计分析 | 模型间的性能差异是否有统计显著性？ |
| 局限性 | 你的方法最大的局限是什么？结论能推广到什么范围？ |
| 应用价值 | 你的研究对实际地貌填图/GIS空间分析有什么实际帮助？ |

## 答辩回答原则

- 每个回答控制在1-2分钟口头陈述
- 有证据（引具体的实验结果/数据支持）
- 不过度承诺（"我们初步发现…在特定条件下…"）
- 每个回答的收尾回到论文的核心贡献

## 地学AI答辩常见攻击点

- "你的方法相比随机森林只提升了几个百分点，值得用深度学习吗？"
  → 先承认幅度 + 强调自动特征学习的额外价值 + 可解释性分析证明学到了合理地形特征 + 未来轻量化方向
- "你的模型只在川东北验证了，能推广吗？"
  → 诚实说明当前验证范围 + 空间交叉验证已评估研究区内稳健性 + 未来跨区域验证计划
