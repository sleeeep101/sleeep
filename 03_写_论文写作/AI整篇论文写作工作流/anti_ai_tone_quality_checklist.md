# 降AI腔质量检查清单

> 目的：让论文更像真实研究过程的产物，而不是模板化的AI输出。
> 原则：不改事实、不改数据、不改引用。只改善表达质量。

## A. 模板化句式检查

- [ ] 论文中没有连续3段以上使用"首先...其次...最后"结构
- [ ] "此外"、"同时"、"另一方面"等连接词不超过总段落数的30%
- [ ] "具有重要的理论意义和实践价值"未出现（或仅出现1次且有具体解释）
- [ ] 没有大段使用"综上所述"、"总而言之"等机械收束
- [ ] "为...提供了科学依据/技术支撑/参考"未滥用（每篇论文最多1次）

## B. 空泛意义检查

寻找并修正以下空泛表述：

- [ ] "具有重要意义" → 改成：对谁有意义？什么意义？用于什么场景？
- [ ] "广阔的应用前景" → 改成：具体可以在哪些领域应用？
- [ ] "为相关研究提供参考" → 改成：为哪类研究提供什么参考？
- [ ] "效果良好" → 改成：具体什么指标达到了什么水平？
- [ ] "表现出明显的优势" → 改成：在XX指标上相比YY方法提升了ZZ%
- [ ] "具有一定的价值" → 删除或量化

## C. 证据链检查

- [ ] 每个"研究表明"后面都有具体引用
- [ ] 每个"结果表明"后面都能对应到具体图表
- [ ] 每个"可能是因为"后面都标注了确定性等级
- [ ] 没有"众所周知"、"广泛认为"等无来源声明
- [ ] 统计数据都有来源（自己的计算或引用）

## D. 段落任务检查

每段检查：这一段的任务是什么？完成了吗？

- [ ] 引言段：每段解决一个问题（背景/进展/空白/本研究目标）
- [ ] 方法段：每段描述一个方法/步骤（原理→操作→参数→原因）
- [ ] 结果段：每段对应一个/一组图表（描述→关键数字→初步解释）
- [ ] 讨论段：每段讨论一个主题（对比→解释→不确定性→意义）
- [ ] 没有"万金油段落"（放在哪章都行的通用描述）

## E. 研究问题对应检查

- [ ] 引言中提出的研究问题在结论中都有回应
- [ ] 方法中描述的每个分析都对应对应一个研究问题
- [ ] 结果中报告的每个发现都与某个研究问题相关
- [ ] 讨论中解释的每个现象都来源于结果
- [ ] 没有"跑题"内容（与研究问题无关的炫技分析或空泛讨论）

## F. 图表结果对应检查

- [ ] 正文中引用的每个图表编号都存在且正确
- [ ] 图表描述中的数字与图表显示的一致
- [ ] 同一数据在不同位置（摘要/结果/讨论/结论）引述一致
- [ ] 没有"如图所示"但未给出图号
- [ ] 没有"结果表明"但没说是哪个图表/哪个统计量

## G. 个人判断与局限性检查

- [ ] 讨论中明确区分了"本研究结果"和"推测"
- [ ] 至少列出3条研究局限性（方法/数据/范围各至少1条）
- [ ] 不确定的结论使用了"可能"、"推测"、"需要进一步验证"等限定语
- [ ] 没有把"本文未研究"写成"本文发现"
- [ ] 结论没有超出研究区/研究方法的适用范围

## H. GIS/遥感/空间分析表达检查

GIS论文特有的AI腔问题：

- [ ] 地形描述有具体数字（范围/均值/标准差），不是只有"复杂""多样"
- [ ] 空间分布描述有具体方向和位置，不是只有"呈现一定的空间分异"
- [ ] 方法选择有GIS领域的理由（数据特性/计算效率/空间自相关等）
- [ ] 分辨率/尺度效应被提及和讨论
- [ ] 投影/坐标系信息在方法中明确
- [ ] 没有把"空间分析"当成黑箱（每个分析步骤都有具体描述）

## I. 检测器高风险句子的质量诊断

如果某句被AI检测器标记为高风险，用以下问题诊断：

1. 这句话有没有具体信息？（数字/地名/方法名/引用）
   - 没有 → 补充具体信息
2. 这句话换一个人写会怎么写？
   - 完全相同 → 模板化表达，需要改写
3. 这句话删掉会丢信息吗？
   - 不会 → 冗余句子，删除
4. 这句话的证据在哪？
   - 找不到 → 需要补充引用或数据
