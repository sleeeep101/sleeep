# 防学术造假检查清单

> 论文提交前逐项检查。任何一项标❌都需要修正后再提交。

## A. 文献真实性

- [ ] 论文中引用的每一篇文献都可以在Google Scholar/CNKI/Web of Science中检索到
- [ ] 每个引用都有对应的DOI（或可通过其他方式验证）
- [ ] 作者名、年份、期刊名与原文一致
- [ ] 文中引用的观点确实来自被引文献（不是"看起来像"就引用）
- [ ] 没有"二手引用"伪装成"一手引用"（说你读了但你只读了引用它的文章）
- [ ] 文献数量合理，没有为了凑数而引用无关文献

## B. 数据真实性

- [ ] 所有数据来源都可以追溯（URL、数据库名称、获取日期）
- [ ] 公开数据的版本号和获取时间已记录
- [ ] 非公开数据有获得途径说明（申请/购买/合作）
- [ ] 数据预处理步骤真实可复现
- [ ] DEM分辨率、遥感影像日期、气象站点数量等参数与实际一致
- [ ] 没有声称使用了某数据但实际未获取

## C. 图表真实性

- [ ] 每张图/表都对应真实的数据处理结果
- [ ] 图表编号在正文中正确引用
- [ ] 图表的标题和说明与图表内容一致
- [ ] 统计图表的误差线和显著性标注来自真实计算
- [ ] 没有从其他来源复制图表但未注明
- [ ] 示意图如非原创已标注来源

## D. 方法真实性

- [ ] 论文中描述的所有分析步骤都是真实执行的
- [ ] 软件版本和参数设置与实际操作一致
- [ ] 如有代码，代码能复现论文中的结果
- [ ] 没有声称使用了某方法但实际只用了简化版
- [ ] 没有为了使结果更好看而事后修改方法描述

## E. 结果真实性

- [ ] 结果中的每个数字都可以追溯到具体的分析输出
- [ ] 没有选择性报告（只报告有利结果，隐藏不利结果）
- [ ] 统计显著性检验真实进行，p值来自真实计算
- [ ] 没有为了达到显著而操控样本量或分析方法
- [ ] 异常值和缺失数据的处理方式已说明

## F. 结论边界

- [ ] 每条结论都有对应的结果支撑
- [ ] 结论没有超出研究范围（如：在A区域的研究结论没有被推广到全国）
- [ ] 确定性表述与证据强度匹配（"可能"vs"证明"vs"表明"）
- [ ] 没有把相关关系写成因果关系
- [ ] 研究局限性已诚实说明

## G. AI使用透明度

- [ ] AI参与写作的范围已记录（哪些部分AI辅助生成）
- [ ] AI检测器的诊断结果已记录
- [ ] 所有AI生成的建议都经过人工审核
- [ ] 没有AI编造的内容进入最终版

## H. 快速红线检查（3分钟完成）

以下任何一条为"是"，立即停止，修正后再提交：

- [ ] ❌ 是否有你不确定是否真实存在的文献被引用？
- [ ] ❌ 是否有你不记得实际获取过的数据被列为数据来源？
- [ ] ❌ 是否有你没有实际执行过的分析出现在方法中？
- [ ] ❌ 是否有图表是你没有从真实数据中生成的？
- [ ] ❌ 是否有统计数字你不确定从哪里来的？
- [ ] ❌ 是否有AI生成的句子你没有逐句检查过？
- [ ] ❌ 是否有结论你的数据和结果实际上支撑不了？
