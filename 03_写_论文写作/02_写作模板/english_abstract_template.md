# English Abstract Template (SCI)

> Structured abstract for SCI journals. ~250 words.

## Formula

```
{Background 1-2 sentences} {Gap}. This study {method} to {objective}
in {study area}. Results show that: (1) {finding1}; (2) {finding2};
(3) {finding3}. These findings {implication/significance}.
```

## Template by Journal Type

### Remote Sensing / GIS journals (RS, IJGIS, ISPRS)

```
{Environmental problem} poses significant challenges to {region/domain}.
Accurate {mapping/monitoring/modeling} of {variable} is crucial for
{application}. However, existing methods {limitation}.

This study proposes a {novel approach} that integrates {data sources}
with {method} to {objective}. The method was applied to {study area}
using {dataset details}. Results demonstrate that the proposed approach
achieves {quantitative result}, outperforming {baseline} by {X%}.
Specifically, (1) {finding1}; (2) {finding2}. The {method} provides
a promising tool for {application domain}.
```

### Soil/Earth Surface journals (Geoderma, Catena, ESPL)

```
Soil erosion by water is a {severity} problem in {region}, driven by
{factors}. While {existing approaches}, knowledge gaps remain regarding
{specific gap}. This paper investigates {research question} through
{method} based on {data}. The results indicate that {main finding},
with {quantitative details}. The findings highlight the importance of
{implication} for {soil conservation / land management / policy}.
```

## Checklist

- [ ] ≤250 words
- [ ] No citations
- [ ] No abbreviations without definition
- [ ] Quantitative results included (not just "improved")
- [ ] Keywords: 5-8, MeSH-compatible
- [ ] Self-contained (reader doesn't need to read the paper)
