## 八维终稿检查

| 维度 | 检查项 |
|------|--------|
| 题名页 | 标题/作者/单位/通讯作者/ORCID → 格式是否符合期刊要求 |
| 摘要 | 字数、中英文是否一致、关键词数量和格式 |
| 正文 | 章节编号格式、图表引用连续、公式编号、符号定义完整 |
| 图表 | 分辨率≥300dpi、颜色模式RGB/CMYK、字体大小统一、地图含比例尺+指北针+坐标系 |
| 参考文献 | 格式统一、DOI完整、双向引用一致、中英文比例合理 |
| 格式 | 页边距/行距/字体/字号/页面行号、LaTeX/Word模板 |
| 声明 | 利益冲突/作者贡献/数据可用性/伦理声明/基金致谢 |
| 附件 | 补充材料与正文引用一致、文件命名规范 |

## 常见致命错误

- 图表编号跳号或重复 → 编辑可能退回
- 参考文献格式不一致 → 审稿人认为不熟悉期刊
- 地图缺少比例尺/指北针/坐标系 → 地学论文直接退修
- 摘要中的数字与结论中的数字不一致 → 可信度归零

## 使用方式

投稿前加载 Prompt #64，逐项勾选检查。预计耗时半天，是性价比最高的投稿前工作。
