# 论文写作质量检查清单

## 提交前必查
- [ ] 无错别字（特别是"的/地/得"、"在/再"、"做/作"）
- [ ] 无语法错误（搭配不当/成分残缺/语序不当/句式杂糅）
- [ ] 标点符号正确（中英文标点不混用、引号/括号匹配）
- [ ] 无口语化表达
- [ ] 核心概念首次出现时已定义
- [ ] 术语前后使用一致
- [ ] 研究问题在引言中已明确
- [ ] 方法选择有理由说明
- [ ] 数据来源和时间已标注
- [ ] 结果有统计检验支撑
- [ ] 讨论不过度推断
- [ ] 结论回答了研究问题
- [ ] 所有引用在正文和参考文献中对应
- [ ] 无"已有研究表明"但无具体文献的表述
- [ ] 图表有编号和标题
- [ ] 地图有指北针/比例尺/图例/坐标标注

## GIS/遥感论文额外检查
- [ ] 研究区范围明确
- [ ] 数据空间分辨率已标注
- [ ] 坐标系统已说明
- [ ] 遥感影像波段组合已标注
- [ ] 分类精度（OA/Kappa）已报告
- [ ] 底图来源已标注
