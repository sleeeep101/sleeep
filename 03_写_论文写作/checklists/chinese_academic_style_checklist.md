# 中文学术风格检查清单

## 口语化检查
- [ ] 无"搞/弄/整/啥/咋/嘛/吧/呗"
- [ ] 无"得了/算了/行了/对了"等口头禅
- [ ] 无过度口语化的副词

## 模板化检查
- [ ] 无"随着……的发展"
- [ ] 无"在……背景下"（除非有具体背景说明）
- [ ] 无"具有重要意义"（替换为具体说明）
- [ ] 无"一定程度上"（给出程度或删除）
- [ ] 无"诸多研究表明"（引用具体文献）
- [ ] 无"学术界普遍认为"

## 过短/过长句
- [ ] 无连续5句以上短于15字
- [ ] 无单句超过80字
- [ ] 句式有变化（不连续3句同构）

## 术语检查
- [ ] GIS/遥感术语使用正确
- [ ] 中英文术语混用时格式正确
- [ ] 缩写首次出现时有全称
