# 写作学习卡片

> Prompt ID: 12 | 用途: 从优秀论文段落提取写作学习卡片

## 输入
一段论文原文（中文或英文）或你认为写得好的句子/段落。

## 输出（每张卡片至少包含）
| 字段 | 说明 |
|------|------|
| 来源 | 论文标题/作者/年份 |
| 原句/原段 | 完整原文 |
| 写作功能 | 这段话在论文中的作用（如"引出研究gap"/"解释方法选择理由"） |
| 句式结构 | 可学习的句式模板（不照抄，填空式） |
| GIS/遥感可迁移表达 | 能否用在 GIS/遥感论文中 |
| 不建议照抄的部分 | 什么不能直接搬用 |
| 我的替代表达 | 用自己的话改写 |

## 禁止
- 一段话拆成多张卡片堆数量
- 写"很棒""很好"等无信息量评价
