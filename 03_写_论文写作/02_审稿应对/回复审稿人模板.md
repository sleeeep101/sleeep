## 三段落回复格式（Response-Revision-Location）

```
Response: 感谢审稿人的建议。我们认同……/原文表述不够清晰，已在修订稿中补充说明……
（1-2句概括立场）

Revision: 具体修改内容——改了什么、在哪里
（不需要重复粘贴修改后的全文，概括做了什么修改）

Location: 修订稿第X页第Y段第Z行 / 新增表X / 修改后的图Y
```

## 核心原则

- 语气礼貌（先感谢审稿人的时间和建议）
- 让审稿人容易核查（不需要反复对比新旧版本）
- 无法完全采纳时诚实说明原因（资源/条件/时间限制+替代方案）
- 避免防御性语言（"审稿人没有理解……"→"原文表述不够清晰，已补充说明……"）

## 使用方式

加载 Prompt #53，提供审稿意见原文+你的修改内容，Claude 生成逐条回复草稿。
