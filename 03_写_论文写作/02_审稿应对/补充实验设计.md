## 最小但有说服力的补充实验

| 审稿关切 | 优先补的实验 | 预估时间 |
|----------|-------------|:---:|
| 基线不够强 | 加入1-2个代表性深度学习模型（Swin-UNet/DeepLabV3+） | 1-3天 |
| 泛化未验证 | 相邻流域外部验证（不需额外标注→用已有分类图或目视评估） | 1-2天 |
| 统计不严谨 | Bootstrap 95%置信区间 + 显著性检验（McNemar/Wilcoxon） | 几小时 |
| 模块贡献不明 | 关键消融实验（移除审稿人质疑的模块） | 1-3天 |
| 错误未分析 | 误分类样本的空间分布图和地形学解释 | 半天 |

## 如果结果不理想

- "虽然补充实验未显示统计显著差异，但……" → 诚实回应
- "结果方向一致但幅度较小，可能受限于……" → 提供合理解释
- "这反而揭示了……的有趣现象，值得未来进一步探索" → 转劣势为研究展望

## 使用方式

加载 Prompt #54，提供审稿意见+现有实验条件+可投入时间，获得最小但精准的补充实验方案。
