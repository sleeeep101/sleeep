# write/ 速查

> write/ 是学术写作辅助子系统，属于 10 板块中的**板块 7 — 论文写作库**。
> 打开后 30 秒内看懂。

---

## write/ 是什么

中文学术论文写作辅助工具集。提供：语法检查、学术润色、文献综述准备、GIS/遥感专用写作风格指导、参考文献检查。

---

## 什么时候用

- 写课程论文、小论文、开题报告、毕业论文时
- 需要检查中文语法和错别字时
- 需要从论文卡片整理文献综述时
- 需要 GIS/遥感方向的学术写作风格指导时

## 什么时候不用

- **研0 入学前** — 没有实际写作需求，不需要用
- 日常论文阅读 — 这是论文阅读库（板块2）的事
- 组会 PPT — 这是组会 PPT 库（板块8）的事
- 代码和脚本 — 这是方法和代码库（板块4）的事

---

## 子目录说明

| 目录 | 干什么 | 什么时候用 |
|------|--------|-----------|
| `scripts/` | 7 个 Python 脚本（语法检查、文献综述构建、写作卡片提取、参考文献检查等） | 执行具体写作任务时 |
| `references/` | 7 个参考文件（写作规范、风格指南等） | 需要查规范时 |
| `prompts/` | 13 个 AI 提示词模板 | 调用 AI 辅助写作时 |
| `checklists/` | 4 个检查清单 | 提交前自查 |
| `templates/` | 3 个文档模板 | 新建论文草稿时 |
| `outputs/` | 写作输出和句式库 | 存放写作产物 |
| `config/` | 配置文件（语法规则、能力映射等） | 需要调整检查规则时 |
| `skills/` | 5 个子 Skill | 按需调用 |

---

## 研0阶段最小用法

**基本不用。** 入学前没有实际写作需求。如果偶尔需要：

1. 写一段文字 → 用 `scripts/check_chinese_grammar.py` 检查语法
2. 需要润色 → 用对应的 prompt 模板

不需要深入学习全部 30+ 个文件。

---

## 入学后使用方式

1. 写课程论文/开题报告 → `QUICK_START.md` → 选择对应流程
2. 每次提交前 → 语法检查 (`check_chinese_grammar.py`)
3. 需要文献综述 → `build_literature_review_pack.py`
4. 每周 → 从论文中提取 3-5 条写作句式 → 写入 `outputs/`
