# 知识卡片规范

## 22字段模板

```
- **作者:** 
- **年份:**  | **来源:**  | **评分:** /100 | **等级:** 
- **主题:** 
- **一句话:** 
- **方法:** 
- **迁移:** 
- **可信度:**  | **加入:** 
- **全文证据状态:** 
```

## 深度要求（2026-07-21强化）

| 分析维度 | 要求 |
|------|------|
| 方法提取 | 至少1个可迁移的方法名称+原理简述 |
| 迁移价值 | 明确写出对自己的研究有何具体借鉴 |
| 可信度评估 | 有代码→高, 有数据→中高, 仅有论文→中, 预印本→中低 |
| 关联性 | 标注与已有KB条目的关系 |

## KB入库格式

```
### KB-YYYY-MM-DD-NN | 论文标题
- **作者:** ... | **年份:** ... | **来源:** ...
```
