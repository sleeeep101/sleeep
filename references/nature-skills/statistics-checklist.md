# 统计报告审查清单

> 来源：nature-statistics skill `references/common-failure-modes.md`
> 用途：论文写作质量核验中「可复现性」维度的统计补充

## 核心三问

nature-statistics 的核心框架：对每项统计声明，分离三个问题：

1. **测量了什么？**（变量/指标定义）
2. **分析单元是什么？**（独立实验单元/生物学重复/技术重复/细胞/视野/模拟运行）
3. **推断声明是什么？**（从什么证据推到什么结论）

## 独立实验单元（n 的定义）

**默认规则**：将独立实验单元作为默认 `n`。以下不应被当作独立样本：

| 误作为独立的 | 正确做法 |
|-------------|---------|
| 细胞计数（来自同一培养皿） | 培养皿/动物 = n |
| 视野/ROI（来自同一样本） | 独立样本 = n |
| 重复读数（同一仪器同一样本） | 技术重复，取均值 |
| 多次模拟运行（同参数） | 标注 random seed，取均值和方差 |
| 光谱/像素（来自同一样本） | 独立样本 = n |

## 常见失败模式快速检查

| 场景 | 风险 | 检查方法 |
|------|------|----------|
| 嵌套数据（样地→样方→重复） | 伪重复 | 确认分析层级与实验设计层级一致 |
| 多重比较（>5 组/多个终点） | 假阳性膨胀 | 确认多重比较校正方法（Bonferroni/FDR/…） |
| 细胞层面测量 | 把细胞当 n | 确认 n = 独立培养/独立动物数 |
| 交互作用声明 | 主效应未报告 | 确认先报告主效应，再报告交互 |
| 相关性分析 | 因果推断 | 确认未从相关性推断因果 |
| 小样本（n<5/组） | 检验假设不满足 | 确认检验前提（正态性/方差齐性）或使用非参数检验 |
| p 值选择性报告 | 仅报显著结果 | 确认所有预设终点都有结果（包括不显著的） |
| "显著" vs "不显著" 比较 | 比较 p 值 | 确认不做"X 显著 Y 不显著 → X > Y"的无效推断 |

## 图注统计规范

每张图的图注应包含：
- 样本量（n = ?）
- 数据呈现方式（mean ± SD / median ± IQR / individual points）
- 统计检验方法
- p 值或显著性标记（*p<0.05, **p<0.01, ***p<0.001）
- 如果用了多重比较校正，标注校正方法

## 统计分析方法小节模板

```
## 统计分析

所有数据分析在 [软件/版本] 中进行。数据以 [mean ± SD / median ± IQR] 表示，
除非另有说明。组间比较采用 [检验方法]，多重比较采用 [校正方法] 校正。
[n] 的定义为 [独立实验单元/独立样本]。效应量以 [Cohen's d/η²/…] 报告。
p < 0.05 认为有统计学意义（[如果用了不同阈值，说明理由]）。
```
