# Figure Contract（制图契约）

> 每张图开工前填写此契约，对齐需求后再动手。避免"画完不是想要的"。

## 基本信息

| 字段 | 值 |
|------|-----|
| 图表编号 | Fig_{N} |
| 图表级别 | T1发表级 / T2汇报级 / T3草稿级 |
| 目标期刊/场合 | |
| 尺寸 | 单栏(8cm)/双栏(16cm)/整页 |
| 数据来源 | |
| 配色方案 | 灰度/彩色/Color Universal Design |

## 图表类型

- [ ] 地图 → 需要：比例尺/指北针/图例/坐标系标注
- [ ] 散点图/折线图 → 需要：R²/p值/趋势线/误差棒
- [ ] 柱状图/箱线图 → 需要：显著性字母标记(a,b,c)/样本量
- [ ] 热力图 → 需要：色带/数值标注
- [ ] 流程图/示意图 → 需要：箭头方向/图例

## 核心信息

```
这张图要传达的唯一核心信息是：
_____________________________________________
```

## 检查清单

- [ ] 读者不看正文也能理解图
- [ ] 所有文字标签≥8pt（发表级≥7pt）
- [ ] 颜色在黑白打印下可区分
- [ ] 坐标轴有标签+单位
- [ ] 误差棒/显著性已标注
- [ ] 数据无重复呈现（同一数据只在一张图出现）
- [ ] 与caption文字一致
