# SCI论文项目模板

> clone此目录启动新项目。替换 `{PROJECT}` 为项目简称。

## 目录结构

```
{PROJECT}/
├── 00_项目说明/          # 研究问题、创新点、实验设计
│   ├── README.md         # 项目总览
│   ├── research_question.md  # 研究问题与假设
│   └── innovation_statement.md  # 创新点陈述(1-2句)
├── 01_数据/              # 原始数据、预处理脚本、数据说明
├── 02_代码/              # 分析代码（按实验组织）
├── 03_图表/              # 论文图表（T1发表级）
├── 04_文献/              # 核心参考文献+笔记
├── 05_论文初稿/          # 论文各版本
└── 06_审计与留痕/        # 实验日志、决策记录
```

## 启动检查清单

- [ ] 研究问题：能否用1句话说清楚？
- [ ] 创新点：属于哪种类型（方法迁移/场景迁移/数据融合/参数改进）？
- [ ] 目标期刊：近2年有类似方法论文章吗？
- [ ] 数据：已获取或可获取？数据量够吗？
- [ ] 方法：有参考实现吗？需要多大改动？
- [ ] 时间：1个月能出初稿吗？

## 论文结构（IMRaD）

1. Introduction — 大背景→具体问题→现有Gap→本研究→创新点
2. Methods — 研究区→数据→方法→实验设计
3. Results — 按实验组织，图表+文字描述
4. Discussion — 与已有研究对比→局限性→启示
