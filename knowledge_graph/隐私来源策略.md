# 知识图谱隐私来源策略

默认构建只处理学术工作流内的学术资料。以下来源默认跳过：

- `Personal-Brain`；
- 桌面笔记；
- 联系人文件。

如确有本机私有知识整理需求，需由操作者明确运行：

```powershell
python -m knowledge_graph build --include-private-sources
```

这会读取本机私有内容，生成的 `kg_data.json` 不应直接上传、共享或作为公开版本的知识图谱。公开发布前应从原始公开资料重新构建，并独立检查节点文本、属性、文件路径及缓存文件。

`--include-private-sources` 只是一次显式授权，不是脱敏机制；它不会自动删除姓名、路径、联系方式或敏感地理信息。
