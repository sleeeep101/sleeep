## 7个Prompt一览

| # | Prompt | 一句话用途 | 使用时机 |
|:---:|------|------|------|
| 1 | 头脑风暴研究选题 | 生成≥10个可投稿级选题（各含问题-方法-验证-贡献闭环） | 方向空白时广撒网 |
| 2 | 研究方向聚焦 | 宽泛方向→清晰主问题（核心变量/对象/数据/方法/指标/结论） | 有初步兴趣方向后 |
| 3 | 研究问题SMART化 | 研究主题→≥5个可测量可执行有时限的问题 | 方向明确后精确化 |
| 4 | 研究假设制定 | 主假设+竞争假设+零假设（各含理论依据/验证方法/证据判断） | 问题明确后建假设 |
| 5 | 创新点挖掘 | 五层面创新诊断（问题/数据/方法/评价/应用）+三版本贡献表述 | 方法有雏形后 |
| 6 | 可行性评估 | 七维度打分（数据/标注/计算/方法/周期/伦理/发表）+降险方案 | 方案确定前最后检验 |
| 7 | 研究边界限定 | "研究什么/不研究什么/为什么不研究/如何防审稿人误解" | 方法部分写作前 |

## 推荐链式使用

```
#1 头脑风暴（广撒网10+选题）
    → #2 聚焦（选出2-3个可行方向）
        → #3 SMART化（精确定义研究问题）
            → #4 制定假设（可检验假设体系）
                → #5 创新点挖掘（与已有研究区分）
                    → #6 可行性评估（现实检验）
                        → #7 边界限定（论文范围锁定）
```

## 使用原则

1. **导师优先**：所有选题产出需经罗明良教授确认，AI只做辅助
2. **信息越具体越好**：每个Prompt使用前尽量填入已知信息（数据规模/研究区/预期方法/限制条件/目标期刊）
3. **多轮追问**：AI产出后用以下追问深化：
   - "请指出这个方案最可能被审稿人质疑的3个地方，并给出预防性回应"
   - "请给出最低可行修改版（只保留最核心的部分）"
   - "这个方向在罗明良老师已有研究体系中处于什么位置？"
4. **保存记录**：每次选题讨论产出保存为此目录下的 `{date}_选题讨论.md`
