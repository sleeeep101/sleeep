## 语言规则

除以下字段保留原文外，**所有内容必须用中文输出**：
- 论文标题（保留原文）
- 作者姓名（保留原文）
- 期刊/会议名称（保留原文）
- DOI / arXiv ID / URL
- 必要时保留的英文术语（第一次出现时附中文解释）

英文术语处理：第一次出现用「中文解释 + 英文原词/缩写」格式，如「数字高程模型（Digital Elevation Model, DEM）」，后续可直接用缩写。

## 使用说明

1. 精读论文 → 填完整卡片（全部中文，仅标题/作者/DOI 保留原文）
2. 泛读论文 → 只填基本信息 + 一句话总结 + 评分（标注"泛读"）
3. 只看摘要 → 标注"仅基于摘要"
4. 暂时看不懂 → 标注"后期再读"
5. 所有卡片必须保留 DOI/arXiv ID/URL，即使当时找不到也留空字段，DOI 不确定写「DOI：未核验」

## 证据标签体系

来源：paper-reading-zh。所有声明必须标注证据强度。

### 信息来源标签

| 标签 | 含义 | 何时使用 |
|------|------|---------|
| 未核验 | 未实际检查/验证 | 用户未提供、自己无法验证的信息 |
| 未找到 | 已搜索但找不到 | 查了 arXiv/Google/S2 但没找到 |
| 已核验 | 已核对原文确认 | 在 PDF/网页中找到对应原文 |
| 论文未说明 | 论文本身没提 | 如复现细节、参数、数据量 |
| 仅基于摘要 | 信息只来自摘要 | 无全文时 |
| 作者预测 | 论文中的预测性声明 | 非实验结论，是作者推测 |

### 数字声明的锚定

论文中出现的具体数字必须标注锚点：
- `[Sec 3.2]` — 方法部分
- `[Table 2]` — 表 2
- `[Fig 4]` — 图 4
- `[摘要]` — 仅在摘要中出现
- `[论文未说明]` — 找不到出处

### 跨论文比较标签

| 标签 | 含义 |
|------|------|
| 口径完全可比 | 相同数据集+指标+实验设置 |
| 口径不完全可比 | 数据集/指标/规模不一致 |
| 口径未核验 | 未确认可比性 |
