## 1. PDF 文本提取

| 项目 | 内容 |
|------|------|
| **触发** | 桌面有 PDF 需要阅读、提取论文文本 |
| **Skill** | `pdf-image-text-extractor` |
| **底层** | `pdf_ingest` 模块（7引擎级联） |
| **前置** | **预筛**（2026-07-13新增）：用 `04_paper_prescreen_filter.md` 做方向匹配+期刊加权→A/B/C/X分级，X级跳过 |
| **输出** | paper.md（提取文本）→ 知识卡片 |
| **别做的事** | 不要跳过提取直接用 Claude 看 PDF（大文件上下文浪费）；不要跳过预筛直接提取（低相关度论文浪费资源） |

## 2. 论文精读与知识卡片

| 项目 | 内容 |
|------|------|
| **触发** | 读论文、分析论文、生成知识卡片 |
| **实现** | Claude 主对话 + Sub-Agent 按主题并行 |
| **标准** | ≥15行11字段完整卡片，7维100分制评分 |
| **输出** | 日报中的知识卡片 + ≥60分入长期知识库 |
| **别做的事** | 不只看摘要就生成卡片；不因评分低跳过高分论文 |
