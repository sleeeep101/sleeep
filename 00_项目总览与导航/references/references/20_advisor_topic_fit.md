# 20_advisor_topic_fit — 导师/课题适配机制

所有论文阅读、方法积累、选题生成都要判断与导师方向的适配度。服务目标是「能在导师指导下顺利毕业 + 形成就业竞争力」。

---

## 一、与罗明良方向适配

### 核心研究领域
DEM、数字地形分析、土壤侵蚀、沟谷、流域、地貌、地形因子、GIS 空间分析、空间建模

### 适配判断（每篇论文/每个选题回答）

| 维度 | 判断问题 | 匹配度 |
|------|---------|--------|
| DEM/地形 | 是否涉及 DEM 或数字地形分析？ | 高/中/低/无 |
| 土壤侵蚀 | 是否涉及土壤侵蚀/水土流失？ | 高/中/低/无 |
| 沟谷/地貌 | 是否涉及沟谷、地貌形态？ | 高/中/低/无 |
| 流域分析 | 是否涉及流域/水文分析？ | 高/中/低/无 |
| 空间分析 | 是否涉及 GIS 空间分析方法？ | 高/中/低/无 |
| 地形因子 | 是否涉及坡度/坡向/曲率/TWI 等？ | 高/中/低/无 |

### 罗老师方向综合适配
```
🟢 高度适配: 至少 3 个维度为「高」
🟡 中度适配: 至少 2 个维度为「高」或「中」
🟠 轻度相关: 至少 1 个维度为「高」或「中」
⚪ 不太相关: 无维度匹配
```

## 二、与覃发超 / 遥感测绘 / GIS 应用方向适配

### 核心领域
遥感、测绘、GIS 应用、自然资源、生态环境、空间数据处理、影像分析、横向项目可行性

### 适配判断

| 维度 | 判断问题 | 匹配度 |
|------|---------|--------|
| 遥感 | 是否涉及遥感影像处理/分类？ | 高/中/低/无 |
| 测绘/GIS | 是否涉及测绘或 GIS 应用？ | 高/中/低/无 |
| 自然资源 | 是否涉及自然资源监测/管理？ | 高/中/低/无 |
| 生态环境 | 是否涉及生态环境评估/监测？ | 高/中/低/无 |
| 空间数据 | 是否涉及空间数据处理/分析？ | 高/中/低/无 |
| 横向可用 | 是否能用于横向/应用型项目？ | 高/中/低/无 |

## 三、与硕士毕业适配

### 判断维度

| 维度 | 绿灯（适配） | 黄灯（需调整） | 红灯（高风险） |
|------|------------|--------------|-------------|
| 数据可获得性 | 公开免费数据 | 需要申请但可能获取 | 无法获取或非常昂贵 |
| 方法可控性 | 成熟方法，可复现 | 方法新但有开源代码 | 方法复杂且无参考实现 |
| 硕士体量 | 1-2 年可完成 | 可能需要 2+ 年 | 博士级别工作量 |
| 论文可写性 | 有清晰章节结构 | 结构可以设计 | 难以形成完整论文 |
| 代码要求 | 基础 Python/GIS 足够 | 需要学但可行 | 需要很强代码能力 |
| 数学要求 | 描述性统计足够 | 需要补基础统计 | 需要高等数学/推导 |
| 中文参考 | 有大量中文论文 | 有一些中文论文 | 几乎没有中文文献 |
| 导师指导 | 在导师核心方向 | 导师有接触 | 导师不熟悉 |
| 毕业确定性 | 稳 | 有一定风险 | 高风险 |

综合评分：
- 🟢 **高毕业适配**：红灯 ≤ 1，绿灯 ≥ 5
- 🟡 **中毕业适配**：红灯 ≤ 2，绿灯 ≥ 3
- 🔴 **低毕业适配**：红灯 ≥ 3

## 四、与企业就业适配

### 判断维度

| 维度 | 问什么 | 是/否 |
|------|--------|------|
| GIS 技能 | 能形成 ArcGIS/QGIS 操作能力吗？ | |
| 编程能力 | 能形成 Python/GEE/ArcPy 能力吗？ | |
| 简历价值 | 能写进简历吗？ | |
| 作品集 | 能变成可展示的作品吗？ | |
| 行业贴近度 | 贴近自然资源/测绘/规划/遥感/数据处理岗位吗？ | |
| 技术栈匹配 | 技术栈匹配企业招聘需求吗？ | |

综合评分：
- 🟢 **高就业价值**：「是」≥ 4
- 🟡 **中等价值**：「是」≥ 2
- 🟠 **就业价值低**：「是」< 2

## 六、使用规则

1. **每篇精读论文** → 必填导师/课题适配度字段
2. **每个选题** → 必填四维适配度
3. **周总结/月总结** → 回顾适配度分布，防止方向跑偏
4. **偏离主线的论文** → 除非就业价值 🟢 或方法迁移价值 ≥ 8，否则不精读
5. **适配度过低的选题** → 放入选题池底，不优先推进
