# 05_knowledge_base_schema — 长期知识库结构

> 最后更新: 2026-07-13
> 评分体系: 7维100分制，入库阈值 ≥60

## 目录结构

```
academic-workflow/01_读_论文阅读与复盘/04_长期知识库/
├── 长期知识库.md            ← 评分≥60论文卡片汇总（2.8MB, ~900+条目）
└── 可能的创新点.md          ← 方法创新点汇总
```

## 日报与长期知识库分工

| 维度 | 日报 | 长期知识库 |
|------|------|-----------|
| 内容 | 全部论文（全量台账+完整卡片） | 评分≥60的高价值论文 |
| 卡片深度 | ≥15行11字段完整卡片 | 精简条目（标题/作者/年份/来源/路径/评分/总评/标签/一句话/方法/迁移/可信度/理由） |
| 准入标准 | 全部论文，不按分数过滤 | 7维100分制评分 ≥ 60 |
| 数量 | 不设上限 | 不设上限 |
| 保留周期 | 长期 | 长期累积，按日期标记 |

## 入库评分（7维100分制）

| # | 维度 | 满分 |
|---|------|:----:|
| 1 | 年份时效性 | 10 |
| 2 | 导师方向相关度（DEM/地貌/侵蚀/GIS/水土保持） | 20 |
| 3 | 数据与研究区清晰度 | 15 |
| 4 | 方法流程完整度 | 20 |
| 5 | 结果与结论可信度 | 15 |
| 6 | GIS/遥感/DEM/空间分析迁移价值 | 15 |
| 7 | 对用户研究生选题或作品集帮助 | 5 |

## 入库阈值

| 评分 | 等级 | 处理 |
|:----:|------|------|
| ≥90 | 优先精读与入库 | 重点研究方向候选 |
| 80-89 | 建议入库 | 方法或案例积累 |
| 70-79 | 选择性入库 | 背景或辅助方法 |
| 60-69 | 谨慎入库 | 有明确用途才保留 |
| <60 | 不入KB | 保留日报记录 |

## 入库格式

```
### KB-YYYY-MM-DD-NN｜论文标题
- **标题:** ...
- **作者:** ...
- **年份:** ...
- **来源:** ...
- **本地全文路径:** ...
- **全文证据状态:** PDF_TEXT_FULL
- **正式学术评分 / 100:** XX
- **总评等级:** ...
- **主题标签:** ...
- **一句话总结:** ...
- **方法:** ...
- **可迁移到 GIS / 遥感 / DEM / 空间分析的点:** ...
- **可信度:** 高/中/低
- **加入理由:** ...
```

## 防膨胀规则

- 每周二次整理
- 低价值重复内容删除
- 论文卡片压缩成方法库
- 句式进入写作库
- 可执行内容进入 AI_Code 任务池
