# 13_cnki_chinese_literature_workflow — 中文论文/CNKI 工作流

## 当前阶段策略

中文论文（知网/万方/维普/学校数据库）由**用户手动下载或提供**。academic-workflow 不自动下载、不批量爬取。

用户可以提供以下任一形式：
- PDF 全文 → 按全文阅读
- 摘要 → 标注「仅基于摘要」
- 题录 → 只做初筛判断

## 后续阶段

等用户可以自己获取中文论文后，工作流支持处理：
- 知网/万方/维普论文
- 中文硕士/博士论文
- 中文综述论文
- 导师推荐论文
- CAJ/PDF 转换后的文本

处理原则：
1. 提供 PDF → 按全文阅读
2. 只提供摘要 → 标注「仅基于摘要」
3. 只提供题录 → 只做初筛判断
4. 中文论文重点提取：中文写作句式、国内案例、数据来源、指标体系、硕士论文可借鉴点
5. 判断能否迁移到 GIS/遥感/空间分析/自然资源/生态环境/城市研究/地貌水文/空间数据工程

**常用源**：CNKI / 万方 / 维普 / 百度学术 / Google Scholar 中文
**检索式结构**：{核心概念} + {方法} + {应用场景}

**示例**：
- `(DEM OR 数字高程模型) AND (土壤侵蚀 OR 水土流失) AND (黄土高原)`
- `(深度学习 OR 卷积神经网络) AND (遥感分类 OR 土地利用) AND (高分影像)`

## 中文关键词扩展

英文关键词 → 中文对应词（不能直接机翻，要用领域标准译法）

| 英文 | 中文标准译法 |
|------|------------|
| digital elevation model | 数字高程模型 / DEM |
| soil erosion | 土壤侵蚀 / 水土流失 |
| watershed analysis | 流域分析 / 水文分析 |
| gully erosion | 沟蚀 / 切沟侵蚀 |
| terrain analysis | 地形分析 / 数字地形分析 |
| land use change | 土地利用变化 / LUCC |
| remote sensing classification | 遥感分类 / 遥感解译 |
| spatial autocorrelation | 空间自相关 |
| geographically weighted regression | 地理加权回归 / GWR |

## 论文筛选流程

1. 检索 → 导出题录（CNKI 支持 EndNote/RIS 导出）
2. 读摘要初筛 → 三类：精读 / 泛读 / 跳过
3. 精读 → 按 04_paper_card_template.md 生成卡片
4. 学位论文注意：硕博论文通常方法描述更详细，但对标期刊论文的创新不足
5. 中文核心：测绘学报、地理学报、遥感学报、地球信息科学学报、武汉大学学报·信息科学版

## 中文论文卡片额外字段

中文论文卡片相比英文增加：

## 从中文论文提取写作句式

中文论文的写作逻辑和英文有区别：
1. 中文更强调"研究意义"和"现实需求"
2. 方法描述相对简洁（但学位论文很详细）
3. 讨论部分通常不如英文论文深入

提取句式时标注"中文论文句式"，单独放入写作句式库。

## 中英文论文分工策略

完整的策略见 `references/21_chinese_english_literature_strategy.md`。本文件只补充中文专用检索和筛选流程。

关键分工原则：
| 目的 | 用哪种 |
|------|--------|
| 学方法 | 英文优先 |
| 学前沿 | 英文优先 |
| 学案例（中国） | 中文优先 |
| 学论文结构 | 中文硕博论文优先 |
| 学中文表达 | 中文期刊论文优先 |
| 学实验设计 | 英文优先 |

每周至少：1 篇英文方法论文 + 1 篇中文应用论文 + 1 篇中文综述/学位论文。
