# 09_output_files — 输出目录与文件命名规范

> 最后更新: 2026-07-13

## 语言规则

除论文标题、作者姓名、DOI、URL、必要英文术语外，**所有输出文件内容默认使用中文**。

- 文件名使用中文
- 日期格式统一 `YYYY-MM-DD`
- 英文术语第一次出现附中文解释

## 实际目录结构

```
academic-workflow/
├── 01_读_论文阅读与复盘/
│   ├── 01_每日论文/
│   │   └── YYYY-MM-DD_论文阅读日报.md          ← 每日日报（含全量知识卡片+台账）
│   ├── 02_论文阅读库/
│   │   ├── 已读论文清单.md                     ← 去重索引
│   │   ├── paper_sources/                       ← 在线检索论文源文件（按日期）
│   │   │   └── YYYY-MM-DD/
│   │   │       └── {Author_Year_Title}/note.md
│   │   └── fulltext_papers/                     ← 桌面/本地PDF归档（按日期）
│   │       └── YYYY-MM-DD/
│   │           ├── 每日论文源数据文件/           ← 入库论文原PDF
│   │           ├── 重复文件_已核验/              ← SHA256/题名重复
│   │           └── 未入库_已检查/                ← 证据不足未入库
│   ├── 04_长期知识库/
│   │   ├── 长期知识库.md                        ← 评分≥60论文卡片汇总
│   │   └── 可能的创新点.md                      ← 方法创新点汇总
│   └── 05_阅读提示词/                           ← 4个prompt文件
├── 03_写_论文写作/
│   └── ...
├── 04_组会PPT/
│   └── ...
└── data/pdf_library/processed/                  ← 中间提取产物（每日清理）
```

## 命名规范

| 文件类型 | 格式 | 示例 |
|------|------|------|
| 论文阅读日报 | `YYYY-MM-DD_论文阅读日报.md` | `2026-07-13_论文阅读日报.md` |
| 长期知识库 | `长期知识库.md` | 追加式，按日期标记 |
| 已读论文清单 | `已读论文清单.md` | 追加式，按年份降序 |
| PDF原文归档 | `fulltext_papers/YYYY-MM-DD/` | 三级目录分级存放 |

## 日报结构规范（2026-07-13 基线）

```
# 论文批量阅读日报
## 一、领域总览
## 二、核心论文知识卡片（精选TOP N）
## 三~六、可迁移方法/中文学术表达/后续方向/价值索引
## 七~十、Agent专题摘要（按主题分组）
## 十一、最终统计
## 十二~十五、完整知识卡片（按主题分组，每篇≥15行11字段）
```
