# 07_writing_templates — 学术写作句式与段落模板

适合文字能力弱、需要固定句式和逻辑的学生。每个模板给：句式 → 场景 → GIS 例子 → 注意事项。

> **跨仓库精进**：本文件覆盖句式层面（L1-L4 训练）。写作前的结构层面方法论（声明-证据映射、逆向大纲、Introduction-Twice、压缩操作、英文散文风格规则）见 `<LOCAL_PATH>`。

## 1. 背景引入

**句式**: "{研究领域}是{学科}的核心问题之一，对{应用场景}具有重要意义。"
**场景**: 论文引言第一段
**GIS例子**: "数字地形分析是地貌学研究的基础工具之一，对流域侵蚀评估和地质灾害预警具有重要意义。"
**注意**: 不要写成"随着...的发展"这种万能废话。

**句式**: "近年来，{技术/方法}的发展为{研究问题}提供了新的解决思路。"
**GIS例子**: "近年来，深度学习技术在遥感影像分类中的快速发展，为土地利用变化检测提供了新的方法论支撑。"

## 2. 研究重要性

**句式**: "{问题}直接影响{后果1}、{后果2}和{后果3}，是{领域}亟待解决的关键问题。"
**GIS例子**: "土壤侵蚀直接影响耕地质量、水库淤积和河道安全，是水土保持领域亟待解决的关键问题。"

**句式**: "准确{动词}{研究对象}对于{下游任务}至关重要。"
**GIS例子**: "准确估算植被覆盖度对于土壤侵蚀建模和生态修复效果评估至关重要。"

## 3. 现有研究不足

**句式**: "尽管{已有方法}在{某个方面}取得了进展，但在{不足之点}方面仍存在{具体问题}。"
**GIS例子**: "尽管 RUSLE 模型在流域尺度土壤侵蚀估算中取得了广泛应用，但在高异质性地形区域的 LS 因子计算上仍存在较大不确定性。"

**句式**: "现有研究主要聚焦于{已有场景}，对{新场景/新条件}下的{问题}关注不足。"

## 4. 本文目标

**句式**: "本文以{研究对象}为例，基于{数据}，采用{方法}，旨在{目标1}、{目标2}，以期为{应用场景}提供{价值}。"
**GIS例子**: "本文以黄土高原典型小流域为例，基于高分遥感影像和 DEM 数据，采用深度学习语义分割方法，旨在实现沟谷地貌的自动识别和演化趋势分析，以期为区域水土流失防治提供空间决策支持。"

## 5. 数据来源描述

**句式**: "{数据类型}来源于{来源/平台}，时间跨度为{YYYY}至{YYYY}，空间分辨率为{X}m，覆盖面积为{X}km²。"

**句式**: "为验证{方法}的{性能/可靠性}，本文收集了{X}个{数据类型}，包括{列举}。"

## 6. 方法介绍

**句式**: "{方法名}的核心思想是{一句话核心}。该方法首先{步骤1}，然后{步骤2}，最后{步骤3}。"

**句式**: "本研究采用的技术路线如图{X}所示：{用文字简要描述流程图}。"

## 7. 实验结果

**句式**: "实验结果表明，{方法A}在{指标1}、{指标2}和{指标3}上分别达到了{X}、{Y}和{Z}，相比{baseline}分别提升了{X%}、{Y%}和{Z%}。"

**句式**: "从{视角}来看，{方法}在{条件A}下表现{好/差}，这可能是由于{可能原因}。"

## 8. 讨论

**句式**: "{某个结果}值得关注。{进一步解释}。这一发现与{引用研究}的结论{一致/不一致}，可能的解释是{解释}。"

**句式**: "需要指出的是，本研究存在以下局限：{局限1}、{局限2}。后续研究可进一步{改进方向}。"

## 9. 结论

**句式**: "本文通过{方法}，研究了{问题}。主要发现包括：(1){发现1}；(2){发现2}；(3){发现3}。"
**注意**: 结论不要重复摘要和引言。

## 10. 中文学术表达常见误区

- ❌ "通过...，使得..." → ✅ "通过...，..."
- ❌ "进行了分析" → ✅ "分析了"（"进行"是弱动词，尽量不用）
- ❌ "对...进行了研究" → ✅ "研究了..."
- ❌ "有着...的作用" → ✅ "可..."或直接描述
- ❌ "表现出较好的性能" → ✅ 给具体数值

## 从论文中提取句式的方法

每读一篇论文，找到 1-2 个写得好的句子 → 分析为什么好 → 抽象成模板 → 用 GIS 话题重写 → 写入写作句式库。

```markdown
### 句式: {功能描述}
- **原句**: {论文中的原文}
- **为什么好**: {逻辑清晰? 简洁? 过渡自然?}
- **模板**: {抽象化}
- **GIS 改写**: {替换内容}
- **来源**: {论文DOI}
```

## 渐进写作训练机制

参照 `references/16_done_definition.md` 写作训练完成标准。四个阶梯：

| 阶段 | 能力 | 训练方式 | 当前状态 |
|------|------|---------|---------|
| L1 填词 | 能套用固定句式 | 给定句式模板，填入自己的研究内容 | ← 从这里开始 |
| L2 仿写 | 能模仿论文段落结构 | 读一段原文 → 用自己的话题写一段 | |
| L3 改写 | 能改进自己或他人的段落 | AI 标记问题 → 自己改写 → AI 再检查 | |
| L4 独立 | 能从头写出合格段落 | 给定主题和要点 → 独立写一段 | |

### L1 填词训练
1. 选择一个句式模板（如「背景引入」）
2. 把模板中的 `{研究领域}`、`{应用场景}` 等占位符替换为自己的研究内容
3. 对比原句，检查逻辑是否通顺

### L2 仿写训练
1. 从论文中选取一段写得好的段落（引言/方法/讨论）
2. 提取其逻辑链：第一句做什么 → 第二句做什么 → ...
3. 用自己的研究话题写一个同样逻辑链的段落
4. AI 对比原文和仿写的差距

### L3 改写训练
1. 自己写一段（可以是之前 L1/L2 的产物）
2. AI 标记：逻辑问题、表达问题、冗余、缺少过渡
3. 自己改写
4. AI 再次检查，直到通过

### L4 独立写作
1. 给定：主题 + 核心要点 + 目标段落类型
2. 独立写一段
3. AI 只评价不修改

### 每周写作训练要求
- 至少完成 1 次 L1 或 L2 训练
- 提取至少 1 个新句式入句式库
- 中文句式从中文论文学，英文句式从英文论文学
