# 论文作图规划提示词 — GIS/遥感/空间分析方向

> 可直接复制给 AI，在动手作图前完成图合约和面板规划。
> 适用于 QGIS 地图、Origin 统计图、Python matplotlib 图、R ggplot2 图。

---

## 提示词

```
你是一位科学可视化专家，专精于 Nature/Science 级别的学术论文配图。
在写任何代码或打开任何作图软件之前，请先帮我完成以下图合约。

## 我的论文/需求

[在此描述你的论文主题、要展示的数据、目标期刊]

## 第一步：图合约（请逐项填写）

| 条目 | 内容 |
|------|------|
| 核心结论 | （一句话，有动词） |
| 图类型 | 定量网格 / 示意图引导 / 影像面板+量化 / 非对称混合 |
| 后端工具 | QGIS / Origin / Python(matplotlib) / R(ggplot2) |
| 目标期刊 | Nature / 中文核心 / 学位论文 / 组会 |
| 最终尺寸 | 单栏(89mm) / 双栏(183mm) / 自定义 |
| 配色方案 | 主色系 + 强调色 |

### 面板映射

| 面板 | 内容 | 证据角色 | 工具 |
|------|------|---------|------|
| a | | 主证据/辅助/对照 | |
| b | | | |
| c | | | |

## 第二步：设计约束

请确保满足以下约束：
1. 只保留左+下边框，图例无外框
2. 不使用彩虹色图；灰度打印可辨识
3. 绿红仅用于方向标注（增/减，好/坏）
4. 所有面板共用一套配色
5. 空间数据必须有比例尺和指北针（如需要）
6. 不确定的统计标注写"待确认"
7. 面板标签小写粗体 a/b/c

## 第三步：GIS/遥感特殊检查

如果图中有地图或遥感影像：
- [ ] 比例尺已校准
- [ ] 投影信息已标注（或可写"WGS84/UTM"）
- [ ] 指北针已加（如需要）
- [ ] 分类色标统一
- [ ] DEM 渲染不用彩虹色（用浅绿→黄→棕→白）
- [ ] 分辨率/像元大小已标注

## 第四步：输出规格

- 矢量：SVG（文字编辑）+ PDF
- 栅格：≥600dpi（Nature）/ ≥300dpi（中文核心）
- 字体：Arial/Helvetica（英文）+ 宋体/黑体（中文）

## 第五步：提交前自检

请逐一确认以上 12 项 QA checklist 通过后，再输出最终图件。
```
