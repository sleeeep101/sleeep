# 毕业要求 — 学位论文（大论文）与发表论文（小论文）

> 最后更新: 2026-07-19
> 来源: 西华师范大学研究生院官网 (yjsy.cwnu.edu.cn) + 地理科学学院 (lr.cwnu.edu.cn) 公开通知
> 适用: 2026级地理学（地图学与地理信息系统方向）学术型硕士，学制 3 年
> ⚠️ 官方文件以研究生院和学院最新发布为准，本文为整理版，入学后与导师确认。

---

## 一、学位论文（大论文）

### 1.1 基本要求

| 项目 | 要求 |
|------|------|
| 字数 | 理科 ≥ **1.5 万字** |
| 纸张 | A4，**双面印刷** |
| 装订 | 180g 黄色皮纹纸封面封底，线装或热胶装订 |
| 参考文献 | GB/T 7714-2015 标准 |
| 正文格式 | 宋体小四，英文 Times New Roman，固定值 20 磅行距，首行缩进 2 字符 |
| 页边距 | 上 2.5 / 下 2.5 / 左 3.0 / 右 3.0 cm |
| 章标题 | 黑体三号居中 |
| 节标题 | 黑体小四居左（如 "1.1"） |
| 中文摘要 | 500–1000 字，3–5 个关键词 |
| 图表 | 图题在图下、表题在表上，黑体五号居中；表格用三线表 |
| 页码 | 摘要/目录用罗马数字，正文用阿拉伯数字 |

### 1.2 论文结构（装订顺序）

1. 封面（中文 + 英文）
2. 中文摘要及关键词
3. 英文摘要及关键词
4. 目录
5. 正文
6. 脚注
7. 参考文献
8. 附录
9. 致谢
10. 原创性声明 + 使用授权声明（须亲笔签名）
11. 在学期间科研情况
12. 学位论文数据集

### 1.3 关键流程与时间线

```
研一（2026.09–2027.06）
  ├── 课程学习（总学分 26-30）
  ├── 确定导师研究方向，阅读文献
  └── 初步选题探索

研二（2027.09–2028.06）
  ├── 2027年9月前后：正式开题 ★
  │     └── ⚠️ 开题距答辩必须 ≥ 12 个月
  ├── 小论文撰写与投稿
  └── 论文主体研究/实验/数据分析

研三（2028.09–2029.06）
  ├── 3月20日前：形式审查 + 预答辩
  ├── 3–4月：学术不端检测（最多 2 次机会）
  ├── 4月：匿名送审（2 位专家，周期约 30 天）
  ├── 5月31日前：答辩 ★
  │     └── 答辩委员会 ≥ 5 名副高以上，主席为校外专家
  │     └── 无记名投票，三分之二以上通过
  ├── 6月5日前：终稿查重 + 材料提交
  └── 6月：学院学位评定分委员会表决 → 授予学位
```

### 1.4 盲审红线

- 盲审分数 **60–69 分** → 重点监控论文，需学院教学指导委员会审查
- 1 份不合格 → 可申请第三方送审
- 盲评全"良好"以上 + 答辩"优秀" → 可参评校级优秀硕士论文

### 1.5 电子版提交

需提交 5 个版本：
1. 完整版 Word
2. 完整版 PDF
3. 匿名版 Word（隐去学校代码、学号、LOGO、致谢、授权声明中个人信息）
4. 匿名版 PDF
5. 仅有中文摘要的 TXT

## 二、期刊发表（小论文）

### 2.1 官方硬性要求

**截至 2026 年 7 月，未在公开渠道找到地理科学学院关于硕士学位授予的强制性论文发表规定。**

各学院依据《西华师范大学博士硕士学位授予工作实施细则（试行）》（西华师大校〔2025〕14 号）制定本学院细则。部分学院（如法学院）已明确要求发表知网收录论文 ≥ 5000 字或主持地厅级项目等，**地理科学学院的细则需入学后向研究生秘书或导师确认**。

### 2.2 评奖评优透露的事实标准

虽然毕业硬性要求未公开，但从奖学金和优秀毕业生评审细则中可以反推学院的**实际期望**：

| 评优层级 | 论文要求 |
|----------|----------|
| 年度优秀研究生 | 专业相关学术论文 **1 篇及以上** + 英语六级 426 分 |
| 院级优秀毕业研究生 | 普刊及以上 **2 篇** + 英语四级 426 分 |
| 校级优秀毕业研究生 | **C 类及以上核心 1 篇**，或普刊 **3 篇** + 英语六级 |
| 省级优秀毕业研究生 | **B 类及以上 1 篇**，或 C 类 **2 篇**，或 C 类 1 篇 + 普刊 2 篇 |

> 这些都是"评优"门槛，不是"毕业"门槛。但导师和学院的实际期望通常不会低于"至少发表 1 篇"。

### 2.3 期刊分级（西华师范大学标准，2021 试行版）

地理学/GIS/遥感领域没有固定期刊名单，按数据库收录情况动态认定：

| 级别 | 认定标准 | 地理/GIS/遥感典型期刊举例 |
|------|----------|--------------------------|
| **A1** | SCI JCR 一区/二区 | *Remote Sensing of Environment*(一区), *ISPRS J. Photogrammetry*(一区), *Int. J. GIS*, *Catena*, *Geoderma* |
| **A2** | SCI JCR 三区 | *Environmental Earth Sciences*, *Natural Hazards*, *J. Geospatial Sci.* 等 |
| **B2** | EI Compendex / **CSCD 来源期刊** | 《地理学报》《地理研究》《地理科学》《第四纪研究》《生态学报》《水土保持学报》《地球信息科学学报》 |
| **C1** | CSCD 扩展版 / **北大中文核心**（非 CSCD 来源） | 《测绘科学》《遥感技术与应用》《地理信息世界》《水土保持研究》 |
| **普刊** | 知网收录、非核心 | 各省级学报、一般学术期刊 |

> 罗明良老师课题组常见发文期刊：《生态环境学报》(CSCD/B2)、《西华师范大学学报(自然科学版)》(普刊/C1)、《地理科学》(B2) 等。

### 2.4 务实建议

1. **底线目标**：毕业前至少发表 **1 篇 CSCD/北大核心期刊论文**（C 类以上），确保满足校级优秀毕业生门槛，也为求职作品集加分。
2. **冲刺目标**：1 篇 SCI 三区/四区（A2/B1）+ 1 篇中文核心（B2/C1），对 GIS 企业技术岗的简历竞争力有质的提升。
3. **署名规则**：本人一作或导师一作+本人二作均被认定，优先争取一作。
4. **投稿周期**：中文核心从投稿到见刊通常 6–18 个月，SCI 更久 → **研二开学前（2027 年 9 月）就应该投出第一篇**。

## 三、其他关键约束

| 事项 | 要求 |
|------|------|
| 英语四级 | ✅ 已通过（≥ 426 分），满足校级优秀毕业生门槛 |
| 英语六级 | ❌ 未通过（考研英语 51 分，需专项准备），优秀研究生/省级优秀毕业生要求 CET-6 ≥ 426 分 |
| 开题-答辩间隔 | ≥ **12 个月** |
| 最长修业年限 | **6 年** |
| 学术不端 | 两个检测环节（送审前 + 答辩后），查重不过直接阻断 |
