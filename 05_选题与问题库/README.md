# 选题与问题库

> 研究规划中枢：空白识别 → 创新分类 → 影响力评估 → 选题决策

## 空白识别（Gap Analysis）

### 方法论

1. **文献系统检索**：确定研究领域 → 检索关键词组合 → 筛选近5年核心文献
2. **知识图谱空白检测**：`python -m knowledge_graph query "<关键词>"` 检查已有覆盖
3. **交叉空白**：两个成熟领域的交叉地带往往是高价值空白区
4. **方法空白**：某一方法在相关领域已成功应用但尚未迁移到目标领域
5. **数据空白**：新数据源（UAV/Sentinel-2/高分影像）尚未被充分利用的研究方向

### 可复核的本地证据筛查

`gap_detector.py` 只在明确指定图谱文件后计算“本地覆盖缺口”；没有加载图谱时会保留结论，不会把“无数据”当作“研究空白”。公开下载后的无个人资料示例可直接运行：

```powershell
python 05_选题与问题库/gap_detector.py "gully erosion" --kg-file knowledge_graph/examples/public_graph.json
```

这个示例只验证命令和保守判定逻辑；它不构成文献综述、创新性证明或真实研究结论。正式选题仍须按上面的系统检索、研究尺度/区域/时期比较和导师资源核验执行。

### 空白评估矩阵

| 维度 | 评分标准 | 权重 |
|------|---------|------|
| 研究热度 | 近3年相关论文数量趋势 | 20% |
| 方法可行性 | 现有技术/数据是否能支撑 | 25% |
| 导师方向匹配 | 与导师研究方向重叠度 | 20% |
| 发表潜力 | 目标期刊近期是否有相关专题 | 20% |
| 个人兴趣 | 能否持续投入1-2年 | 15% |

## 创新分类（Innovation Taxonomy）

| 类型 | 说明 | 示例 | 影响力 |
|------|------|------|--------|
| **方法迁移** | 将A领域方法应用到B领域 | GNN从社交网络→沟蚀空间建模 | 中高 |
| **场景迁移** | 同一方法在不同地理区域验证 | 黄土高原沟蚀模型→西南紫色土区 | 中 |
| **数据融合** | 多源数据联合解决单一数据不足 | DEM+Sentinel-2+GEDI森林高度 | 高 |
| **自动化流程** | 手工→脚本→工具链 | QGIS自动化制图/PDF批量提取 | 中 |
| **参数改进** | 改进模型参数或算法 | RUSLE中C因子的遥感反演替代 | 中低 |
| **理论创新** | 提出新概念/新框架 | 极难，硕博阶段不鼓励以此为主 | 高（风险大） |

## 影响力公式（Influence Formula）

```
影响力 = 方法新颖度 × 0.3 + 数据独特性 × 0.2 + 问题紧迫性 × 0.25 + 可复现性 × 0.15 + 导师匹配度 × 0.1
```

每项1-10分，≥7分为高影响力选题。

## 研究规划模板

### 硕士论文选题（2-3年）

```
研究方向: [沟蚀/滑坡/水文/...]
核心方法: [GIS/遥感/ML/DL/统计]
研究区域: [具体流域/行政区域]
数据来源: [DEM/卫星影像/野外实测/公开数据集]
预期产出: [1-2篇中文核心 + 1篇SCI]
关键风险: [数据获取/方法可行性/时间]
```

### 检查清单

- [ ] 与研究领域近3年Top10论文对比，确认不是重复工作
- [ ] 导师同意方向并确认资源可支撑
- [ ] 核心数据可获取（已有或可采集）
- [ ] 目标期刊在近2年内发表过类似方法论的文章
- [ ] 有明确的创新点陈述（1-2句话能说清楚）
