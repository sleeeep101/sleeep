# 04 组会PPT

## 负责什么
组会材料准备、PPT 生成、讲稿撰写、模板管理

## 放什么
- 组会汇报 PPT 文件
- PPT 模板
- 讲稿和发言提纲
- PPT 生成/整理脚本
- 组会相关 prompt 和检查清单

## 不放什么
- 论文阅读内容 → 01_读_论文阅读与复盘
- 论文草稿 → 03_写_论文写作
- 实验数据 → 06_数据资源库

## 核心脚本
- `00_core_scripts/` — PPT 生成脚本 (原 组会/scripts/)
- `03_PPT模板/` — PPT 模板
- `prompts/` — 组会准备提示词
- `speaker/` — 讲稿子系统

## 与其他板块的边界
- 本板块从 01_读、02_作图、03_写 获取内容
- PPT 终稿保存到具体项目文件夹
- 组会后反馈沉淀到 01_读/03_论文复盘
