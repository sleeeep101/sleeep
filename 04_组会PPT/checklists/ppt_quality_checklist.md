# PPT 质量检查清单

> 做完 PPT 后逐页检查。任何一项不通过 → 修改。

## 全局检查

- [ ] 总页数 10–15 页（不超过 20 页）
- [ ] 逻辑线清晰：背景 → 进展 → 问题 → 计划 → 讨论
- [ ] 字号统一（标题 ≥ 28pt，正文 ≥ 18pt）
- [ ] 同一字体（中英文各一种）

## 逐页检查

- [ ] **每页一个核心观点** — 标题是否表达结论？（不是"实验结果"，是"方法 A 优于方法 B"）
- [ ] **正文 ≤ 7 行** — 超过就拆页或删减
- [ ] **每行 ≤ 20 字** — 超过就改成短语
- [ ] **有图表必有解释** — 图表标题是结论？坐标轴有标签？来源标注？
- [ ] **无纯文字页** — 如果全是文字，加一个示意图或流程图

## 内容检查

- [ ] 研究问题在第一页就讲清楚了
- [ ] 本周目标明确（不是"继续研究"）
- [ ] 问题和困难没有隐瞒
- [ ] 尝试过的方案列了（即使失败了）
- [ ] 下一步计划有具体时间线
- [ ] 讨论问题 2–3 条，具体可回答
- [ ] 数据有来源标注

## 视觉检查

- [ ] 图表清晰可读（不是从论文里截的模糊小图）
- [ ] 配色一致（不用超过 3 种颜色）
- [ ] 没有多余动画
