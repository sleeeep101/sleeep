# 组会后复盘清单

> 组会结束后当天填写。不是批评自己，是提取可行动信息。

## 导师反馈

- [ ] 导师说了什么关键的话？（至少 1 条）
- [ ] 导师建议了什么方向/方法？（如有）
- [ ] 导师有没有提出我之前没意识到的问题？

## 同伴收获

- [ ] 谁的汇报和我的方向有关联？
- [ ] 谁用了什么方法/工具是我可以学的？
- [ ] 谁踩了什么坑是我也可能遇到的？

## 自我评估

- [ ] 我这次的 PPT 逻辑通顺吗？（自评 1–5）
- [ ] 我讲清楚了吗？（自评 1–5）
- [ ] 导师听懂了我的问题吗？（自评 1–5）
- [ ] 我最不满意的地方是什么？
- [ ] 下次最想改进的是什么？

## 下周行动

- [ ] 把导师的建议转化成具体任务（3 条以内）
- [ ] 如果有需要联系的人/资源 → 今天发消息
- [ ] 如果有需要读的论文 → 加入 academic-workflow 待补读队列

## 长期跟踪（每月一次）

- 这个月的组会我进步了吗？（表达 / 逻辑 / 问题意识）
- 导师给的反馈变具体了吗？（说明你提的问题变好了）
