# 讲稿检查清单

> 用 ppt-speech-writer 生成讲稿后逐项检查。

## 全局

- [ ] 讲稿语言一致（全中文或全英文，术语除外）
- [ ] 总时长在预期范围内（通常 10–20 分钟）
- [ ] 不是逐字念 PPT — 讲稿是对 PPT 的补充，不是复读

## 逐页

- [ ] **开头是判断句** — 不是"这一页展示了..."，是直接说结论
- [ ] **每页 45–120 秒** — 没有哪页要讲 3 分钟以上
- [ ] **图表有口头解释** — "横轴是 X，纵轴是 Y，可以看到..."
- [ ] **过渡句有** — 每页结束时有一句话衔接到下一页
- [ ] **术语首次出现时有解释** — 不是所有人都知道你的领域缩写

## 开场 2 分钟

- [ ] 研究问题一句话讲清
- [ ] 为什么重要一句话讲清
- [ ] 本周做了什么一句话预告

## 结尾

- [ ] 明确说出"我需要大家的建议/帮助"
- [ ] 具体说出需要什么（2–3 条）
- [ ] 感谢并开放讨论
